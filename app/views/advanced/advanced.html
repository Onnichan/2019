Search tips

  • Seleccione "coincidir con todos" para requerir todos los campos. (por ejemplo, título y autor)
  • Seleccione "coincidir con cualquiera" para encontrar al menos un campo. (por ejemplo, título o tema)
  • Combine palabras clave y límites para encontrar elementos específicos.
  • Utilice comillas para buscar como una frase. (La separación de palabras todavía se aplicará a las palabras dentro de la frase).
  • Usa "+" (sin espacio) antes de una palabra para hacerla necesario.
  • Use "-" (sin espacio) antes de una palabra o frase para excluir.
  • Use "OR", "AND" y "NOT" (mayúsculas) para crear la lógica Boleana dentro de un campo. Puedes usar paréntesis en tus expresiones complejas.
  • Utilice "*" para buscar un término truncado. (La obtención de palabras se realiza automáticamente.
  • Utilizar "?" como un comodín para reemplazar un carácter en una palabra.
  • La derivación de palabras incluye automáticamente formas plurales y singulares, y sufijo común y variaciones tensas.
